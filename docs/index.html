  <h1>ScrapNet — Waste Classification</h1>

  <p>

    This project classifies an input image into one of 7 waste categories:

    <b>cardboard, compost, glass, metal, paper, plastic, trash</b>.

  </p>


  <div class="card">

    <h2>Quick Results</h2>

    <ul>

      <li>Model: EfficientNet-B0 (transfer learning)</li>

      <li>Metrics: accuracy + per-class precision/recall/F1</li>

      <li>Artifacts saved: training curves + confusion matrices + model weights</li>

    </ul>

  </div>
